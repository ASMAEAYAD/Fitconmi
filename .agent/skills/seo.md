# SEO Skill for AI Agents

## Rules
- Always add proper meta title and description to every page
- Title format: "Page Name | FitConMi" (max 60 chars)  
- Description: 150-160 chars, include main keyword
- Add Open Graph tags (og:title, og:description, og:image)
- Use semantic HTML (h1, h2, h3 hierarchy)
- Alt text on every image
- Canonical URLs on every page
- Structured data (JSON-LD) for fitness/health content
